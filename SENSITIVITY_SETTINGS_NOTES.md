# v0.17.13 — 급등·급락 감지 설정

패널에 `애니메이션 감지` 설정을 추가했습니다.

- 기준 시간: 15초 / 30초 / 60초 / 65초 / 2분 / 5분
- 상승 감지: 기본 +0.12%
- 하락 감지: 기본 -0.12% (화면에는 양수 크기로 입력)
- Idle 복귀 기준: 기본 0.07%

기존 캐릭터 이미지, 키프레임, 이펙트 3종 선택 방식은 변경하지 않았습니다.
설정 변경 시 가격 버퍼를 초기화하여 서로 다른 기준 시간의 데이터가 섞이지 않도록 했습니다.
Idle 복귀 기준은 상승·하락 감지값보다 최소 0.01% 작도록 자동 보정됩니다.
