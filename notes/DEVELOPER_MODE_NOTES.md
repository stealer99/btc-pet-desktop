# Developer mode (0.17.11)

- 트레이 우클릭 메뉴에서 **개발자 모드**를 켜면 패널에 테스트 영역이 표시됩니다.
- Idle / Pump / Dump / 봉 갱신 상태를 실시간 가격과 무관하게 강제로 확인할 수 있습니다.
- 차트 이펙트 3종(캔들 3개, 단일 캔들, 로켓/번개)을 패널에서 전환하고 마지막 Pump/Dump 상태로 즉시 재시험합니다.
- 테스트 명령은 IPC 이벤트로만 전달되며 설정 파일에 테스트 상태를 저장하지 않습니다.
- 기존 캐릭터 이미지, 키프레임, 애니메이션 시간값은 변경하지 않았습니다.
